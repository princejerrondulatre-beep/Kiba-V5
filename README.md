KIBA V5

So I made this dash script. Basically, when you do a certain dash animation (the one with ID 10503381238), it automatically sends you flying toward the closest enemy nearby. No more missing your dashes.

What it does:

· Detects when you dash and locks onto the nearest target
· Freezes your movement for a split second so the dash actually connects
· Has a cooldown so you can't spam it (you can change it from 1 to 7 seconds)
· You can turn it on/off with a keybind (default is K, but you can change it in settings)

The GUI:

I went a little overboard with the visuals ngl. It's got this space/galaxy theme with:

· Rotating gradients that look like nebulas
· A little shooting star that flies across sometimes
· Rainbow borders that cycle through colors
· A cooldown bar that fills back up
· A toggle button that pulses when it's on

You can drag the panel around and there's a settings button (the ⚙️) where you can change the cooldown time and the toggle key.

How to use:

1. Run the script
2. The panel shows up on your screen
3. Just play normally and do your dash move — it'll trigger automatically
4. Press K if you want to turn it off for a bit

Notes:

It works in most games that use normal Roblox characters. The cooldown setting saves until you change it again. If the dash animation ID is different in your game, you'd have to edit that part of the code.
